# Examples

This directory contains example scripts demonstrating various features of snackPersona.

## Available Examples

### media_dataset_example.py

Demonstrates how to use the media dataset interface and diversity evaluation features.

**What it shows:**
- Creating a MediaDataset programmatically
- Adding MediaItem objects to the dataset
- Saving and loading datasets from JSON files
- Evaluating diversity of reactions with multiple metrics
- Using the sample media dataset

**How to run:**
```bash
cd /home/runner/work/snackPersona
PYTHONPATH=/home/runner/work/snackPersona python3 snackPersona/examples/media_dataset_example.py
```
